## Usage

### Reality Fork

```
I'll solve this using Reality Fork:

Approach A: Quick fix (5 min)
Approach B: Robust solution (15 min)  
Approach C: Refactor properly (30 min)

<quantum>
  <fork approaches="3">
    [Execute all in parallel, select best]
  </fork>
</quantum>
```

### Knowledge Teleportation

```
<quantum>
  <teleport target="openfang.devops-lead">
    <context>
      Current task: Debugging deployment issue
      Status: Build failing on Render
      Error: Module not found
    </context>
  </teleport>
</quantum>
```

### Agent Swarm

```
<quantum>
  <swarm agents="[coder, tester, reviewer, doc-writer]">
    <task>Implement user authentication system</task>
    <split>
      - coder: Write auth logic
      - tester: Write test cases
      - reviewer: Review approach
      - doc-writer: Write API docs
    </split>
  </swarm>
</quantum>
```

## CLI Commands

```bash
# Check quantum status
openfang-quantum status

# Enable/disable quantum mode
openfang-quantum enable
openfang-quantum disable

# Sync skills with OpenFang
openfang-quantum sync

# Deploy agent swarm
openfang-quantum swarm "Build feature X"

# List MCP servers
openfang-quantum mcp list

# Start MCP servers
openfang-quantum mcp start
```
