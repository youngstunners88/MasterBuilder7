## Memory System

Each agent has:
- **SOUL.md** - Core identity, ethics, personality
- **AGENTS.md** - Decision rules, protocols
- **TOOLS.md** - MCP tool conventions
- **Shared Memory** - Knowledge commons

## Quick Start

```bash
# Clone and setup
git clone https://github.com/youngstunners88/EliteSquad.git
cd EliteSquad
bun install

# Deploy with full squad
bun connector.ts deploy https://github.com/user/repo capacitor 100

# Check status
bun connector.ts status
```

// The squad runs on Zo Computer

## Why 8 Agents?

**Quality over Quantity:**
- Each agent has deep specialization
- Shared memory = collective intelligence
- 3-verifier consensus prevents hallucinations
- Hard spend limits protect budget

**vs. 333 Generic Agents:**
- вқҢ Chaos, coordination nightmare
- вқҢ High compute burn
- вқҢ No memory, no learning
- вң… Focused expertise
- вң… Shared consciousness
- вң… Cost-controlled
- вң… Continuous evolution
